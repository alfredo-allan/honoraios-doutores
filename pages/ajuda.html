Guia de Uso da Aplicação de Gerenciamento de Honorários Médicos

Este guia tem como objetivo explicar o funcionamento da aplicação de gerenciamento de honorários médicos, detalhando cada parte da navegação e funcionalidades disponíveis.

Menu de Navegação: Home Confirmados Não Confirmados

Página Home

A página Home é o ponto de partida da aplicação. Nela, você encontra um modal onde poderá preencher um formulário para cadastrar pacientes como "Confirmados" ou "Não Confirmados". Essa funcionalidade permite organizar os pacientes de acordo com o status de confirmação.

O formulário solicita os dados principais do paciente, como nome, telefone, hospital, lado da articulação, entre outros, para registrar o atendimento médico.

Link "Confirmados"

Na página Confirmados, você tem acesso a uma lista de todos os pacientes que já foram confirmados. Algumas funcionalidades importantes disponíveis nesta página são:

  • Listagem de Pacientes Confirmados: Exibe os dados de cada paciente confirmado, incluindo nome, telefone, data, hospital, etc.
  • Filtro de Pesquisa: Na parte superior da página, você pode filtrar os pacientes confirmados digitando o nome no campo de busca. A lista será atualizada automaticamente com base no que você digitar.
  • Deletar Paciente: Ao lado de cada paciente, existe um botão "Deletar" que permite remover o registro de um paciente, caso seja necessário.

Link "Não Confirmados"

A página Não Confirmados exibe a lista dos pacientes que ainda não foram confirmados. Para cada paciente não confirmado, há um botão de confirmação que, ao ser clicado, atualiza o status do paciente para "Confirmado". Isso facilita o gerenciamento de pacientes, permitindo que você atualize seu status sem precisar preencher novamente o formulário.

Considerações Finais

A aplicação foi desenvolvida para facilitar o gerenciamento dos honorários médicos de forma prática e eficiente. O processo de confirmar, filtrar e excluir pacientes é simples e direto. Siga as instruções acima para navegar pelas páginas e utilizar os recursos disponíveis.
